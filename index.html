    Structure & Tags

    Tag Name Description
    <head> Head Contains metadata about the page. Doesn't display it.
    <title> Title The text displayed on the website's tab on the browser
    <header> Header For intro contents h1 to h6
    <nav> Navigation Bulk of content of webpage.
    <footer> Footer Contains contact info, copyright, terms of use, etc.
    <main> Main Contains metadata about the page. Doesn't display it.
    <body> body What the user can see. Any markup with the content of the page
    <section> Section Divide in specific section for clear reading. Chapters, heading, etc.
    <article> Article Understant type of content. Mostly for screen readers.
    <aside> Aside Info. Adiitional information, not required to understand. Like end notes, comments, etc.
    <figure> Figure Encapsulate media like images, diagrams, etc.
    <figcaption> Figure caption Goes inside figure. to describe the media in the figure tag. → if we move the image, the caption moves with it.
    <img> Image Embed an Image. Can use src with link, or follow the path to file if in the same folder as code.
    <audio> Audio Element To embed audio element. use src and type audio.mp3
    <video> Video Embed a video. Useful attributes: controls, autoplay, loop. *Can also have width and height here.
    <h1-h6> Headings h1-h6 Headings, most important h1, to least h6.
    <p> Paragraph Paragraph Text
    <span> Span In-line container. Works to separate text within another element like p. And give it a class or id.
    <em> Emphasis Style text as italic
    <strong> Strong Importance Style text as bold
    <br> Line Break Create a line break between elements.
    <blockquote> type, reversed, and start type="A", or "I" or "1", and start can be any point on the list
    <hr> Thematic break Creates a line between topic. It's to separate stories. *Shows as a horizontal line
    <ul> Unordered list bullet point list
    <ol> Ordered list numbered point list
    <il> List elements Write elements within a list
    <dl> Description list Sets like question to answer list
    <dt> Description element 1 Like the question or title in the description list
    <dd> Description element 2 The answer or body of the element 1 of the description list
    List attributes type, reversed, and start type="A", or "I" or "1", and start can be any point on the list
    <a> Anchor Elements Link elements outside of page add href, no closing tag. Write id="name", call it with a href="#name"
    <button> Button Element Several types --> submit, radio, checkbox, text...
    <var> for Variables use vas to add a variable X, then sub or sup (insede var) for exponents

    Table Tags & Attributes

    Tag Name Description
    <table> Table Create table
    <thead> Table head Table headings
    <tbody> Table body Include all data minus the heading.
    <tfoot> Table footer For the table footer elements
    <th> Table heading go inside a row in thead, are the headings of the table
    <tr> Table Row Create a row.
    <td> Table data element Creates a data element, that shows.
    Scope: "col" or "row" Heading of Column or Row Gives a colum or row the attribute of head.
    Spanning Columns Spanning rows or columns To spread the colum or row more than 1 box --> colspan="2" or rowspan="2"

    Form Elements

    Tag Name Description
    <form> Create Form Syntax: <form action="#.html" method="GET">
    <label> Label (o input) *Must connect them with input with same ID. Syntax: <label for="meal">What do you want to eat?>
    <input> Input (of label)
    • Syntax: <input type="text" id="meal" name="meal" value="breakfast">
    • Types: text, password, number
    • Attributes:
    • Step: to increase or decrease numbers within the field. step=”1”
    • Range: slider of numbers to choose from. Has min and max.
      • <type="range" min="0" max="100" step="1">
    • Checkbox: Use the same name for all options, but each has a unique label and ID.
    • Radio: show multiple options, and let users choose one or more. *give them the same name to put options together.
    • Dropdown list: create a select attribute and within it create by writing:
      • <select id="lunch" name="lunch"> <option value="pizza">Pizza</option></select>
    • Textarea: bigger text field
      • <textarea id=”blog” name=”blog” rows=”5” cols=”30>Text Goes here</textarea>
      • Set text length, like Twitter: minlength=”5” maxlength=”250”
      • Regular expression: to make sure the content the user inputs follows specific guidelines Ex: pattern=”[0-9] {14-16}”
    • Submit: <input type=”submit” value=”send”>
    <select> Select To create dropdowns. Add after the label, and add all <option> items inside
    <datalist> Datalist Shows a list of options, but not as a dropdown as select. Add all data <option> items inside.
